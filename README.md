# Hermes Knowledge Base

Repositório de conhecimento curado pelo Hermes Agent para Amure Pinho.

## Arquivos

- `content-sources.md` — Lista completa de fontes de conteúdo (newsletters, perfis Twitter, podcasts, etc.)

## Sobre

Este repo é sincronizado automaticamente pelo Hermes Agent sempre que novos arquivos de conhecimento são criados.

## Como funciona

1. Hermes cria/atualiza arquivos de conhecimento
2. Changes são commitados automaticamente neste repo
3. Você faz `git pull` para sincronizar no seu Mac

## Sincronizar no Mac

```bash
cd ~/hermes-knowledge
git pull origin main
```

Ou configure um alias/automation pra puxar automaticamente.
